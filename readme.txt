Для роботи потрібен Python 3.11, 10-11 Win та браузер Edge(якщо використовуєте інший браузер повідомте, також є можливість додати вибір браузеру в config за бажанням)

Встановити необхідні бібліотеки за допомогою команди:
pip install -r requirements.txt
(відкрити термінал в дерикторії де знаходиться файл "requirements.txt" та виконати команду)

Налаштування конфігу
Встановити кукі, якщо це необхідно як отримати кукі вказано на відео:
https://www.youtube.com/watch?v=xUbIqrO5feo

Створити бота за допомогою Bot Father
Отриманий токен помісти в параметер BOT_TOKEN

Налаштування time_settings
'Message Delay': 1, - затримка відправлення повідомлень в телеграм, (впливає на роботу парсеру, чим вона довша тим менша вірогідність проблем з блокуванням)

'Refresh Delay': 20 - затримка перезавантаження сторінки

Для наляглядності записав відео

Запуск програми
Відкрити термінал перемістись в папку з файлами скрипту виконати команду
python main.py

Алгоритм роботи парсеру "FB_parser.py"
Скрипт отримує лінк з вказаними параметрами від користувача з телеграм бота
Прогортає сторінку до тих пір поки прогружаються нові елементи 
Всі елементи які знайдено на сторінці надсилаються в телеграм
Якщо, нові елементи при прогортанні не знайдені то скприпт перезавантажує сторінку
І буде її перезавантажувати за заданим інтервалом(Refresh Delay) до тих пір поки не буде отримано нових елементів яких ще не було надіслано в телеграм

Демонстрація роботи, також продемонстрував роботу двох парсерів одночасно
https://www.youtube.com/watch?v=237TwCAErF4

Є ідея додати функцію де буде проводитись парсинг сторінки з всіма автомобілями
Де параметри ви вказуєте через Telegram бота
Бот проходиться по всій сторінці, в телеграм надсилає леше ті автомобілі які підходять за вказаними параметрами
Так можна уникнути проблем з зупинкою при роботі з багатьма парсирами як зараз
Вартість додавання такої функції з окремою кнопкою в панелі боту становить 750 грн
